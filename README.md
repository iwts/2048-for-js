# 什么是2048？
&#160; &#160; &#160; &#160; 2048是一款益智类游戏，主要通过上下左右的滑动，将相邻的2个一样数字的方块拼合在一起并得到累加值。4*4的方格被完全占满之后，将结束游戏。起始方块有2块，所以每块方块的值一定是2的幂。同时当值为2048时，删除这个方块。
